# Phishing Email Detection Model

## Overview
This project uses Machine Learning to classify emails as **Phishing** or **Safe** based on their content.

## Features
- Text cleaning and preprocessing
- URL detection
- TF-IDF feature extraction
- Logistic Regression model
- User input prediction

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python model.py

3. Enter an email message to get prediction

## Dataset
Dataset is not included due to size limits.

Download from:
https://www.kaggle.com/datasets/subhajournal/phishingemails

After downloading, place it in the project folder and rename it to:
dataset.csv
